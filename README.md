# Digital Receipt Organizer

## Overview
The Digital Receipt Organizer is a web-based application that allows users to upload, store, categorize, and analyze their receipts. The system will eventually include OCR (Optical Character Recognition) to automatically extract vendor names, dates, and amounts from uploaded receipts.

This project is being developed for CIDS 484 as a semester-long  project.

## Current Progress (Milestone 1)
- Repository created
- Project outline added
- Flask project structure created
- Basic file upload prototype working
- Next steps: connect database, build receipt list page, and create dashboard
